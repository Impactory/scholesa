## Remediation Details
For each finding with a `Fixed` status, detail the resolution.

### Finding ID: 001
- **Root Cause**: The API base URL was hardcoded in the service class, making it difficult to switch between different environments (dev, staging, prod).
- **Fix Applied**: Introduced an `EnvironmentConfig` class that reads the URL from a compile-time variable using `String.fromEnvironment`. The `ApiService` was updated to use this configuration, removing the hardcoded value.
- **Verification**: The application can now be compiled with different API endpoints using the `--dart-define=API_URL=...` build flag.
