# Scholesa COPPA Master Alignment

Date: 2026-02-23

This document maps COPPA statutory requirements to Scholesa technical and operational controls.

## 1. School Consent Model
Scholesa operates under the School Consent Exception.
Schools act as parent agents for educational use only.
No behavioral advertising.
No commercial resale of student data.

## 2. Data Collection Disclosure
Documented in Parent Privacy Notice.
Only necessary educational data collected.

## 3. Parental Rights
Parents may:
- Review student data
- Request deletion
- Revoke consent via school

## 4. Data Minimization
AI interactions are grade-band restricted.
Only minimum required data sent to AI providers.

## 5. Security Safeguards
- Encryption at rest (GCP managed)
- TLS in transit
- Role-based access control
- Tenant isolation enforced

Defined in DATA_RETENTION_SCHEDULE.md
